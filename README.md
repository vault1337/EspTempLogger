# EspTempLogger
Arduino based ESP8266 + DHT22 + SSD1306 OLED sensor node.

# Required hardware
- ESP8266
- DHT22
- SSD1306 128*32 pixel OLED display

# Required libraries
- Of course, install ESP8266 boards to your Arduino IDE
- DHT sensor library (Adafruit)
- Adafruit Unified Sensor
- Adafruit SSD1306
